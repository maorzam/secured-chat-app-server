
# Secured Chat App

Secured-Chat-App is an application for encrypted conversation between several participants in a chat room.


## Author

- [@maorzam](https://www.github.com/maorzam)


## Run Locally


Go to the project directory

```bash
  cd Secured-chat-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
